이름 재료 타이핑 (영문)
피셔타이거 반더데켄 + 핸콕 + 파이러츠도킹5 + 레일리 피셔타이거조합 (tiger)
반더데켄 반더데켄 + 검은수염 + 아론 + 해적선 반더데켄조합 (decken)
이완코브 이완코브 + 크로커다일 MR.0 + 버기 마기탄 이완코브조합 (ivankov)
사보 마인 오즈 + 로우 + 사보 사보조합 (sabo)
코알라 징베 + 비비 + 로빈 + 확장팩 코알라조합 (koala)
킨에몬 킨에몬 + 조로 현상금 사냥꾼 + 키드 초신성 킨에몬조합 (kinemon)
캐럿 쵸파 혼포인트 + 에넬 + 압살롬 * 2 + 레일리 캐럿조합 (carrot)
레베카 바제스 + 슈가 + 확장팩 레베카조합 (rebecca)
아오키지 아오키지 + 다이아몬드 죠즈 + 모몬가 아오키지조합 (aokiji)
아카이누 아카이누 + 반더데켄 + 킬러 + 스쿼드 아카이누조합 (akainu)
료쿠규 류마 + 이완코브 + 바르토로메오 + 바솔로뮤 쿠마 료쿠규조합 (ryokugyu)
베르고 상디 디아블잠브 + 바르토로메오 + 스모커 + 로우 베르고조합 (vergo)
검호 미호크 미호크 + 샹크스 + 다이아몬드 죠즈 검호조합 (mihawk)
킬러 캡틴 키드 + 헤르메포 + 킬러 + 바질 호킨스 킬러조합 (killer)
봉쿠레 핸콕 + 봉쿠레 + 버기 마기탄 + 이나즈마 혁명군 봉쿠레조합 (bonkurei)
류마 조로 현상금 사냥꾼 + 류마 + 브룩 류마조합 (ryuma)
비의 시류 시류 + 벤베크만 + 마젤란 시류조합 (shiryu)
페로나 모리아 + 미호크 + 압살롬 + 저격왕 + 페로나 페로나조합 (perona)
써니호 캡틴 키드 + 카쿠 + 해적선 써니호조합 (sunny)
모비딕호 다이아몬드 죠즈 + 마르코 + 에이스 + 해적선 모비딕호조합 (mobydick)
발라티에 제프 + 루피 기어세컨드 + 해적선 발라티에조합 (baratie)
방주맥심 와이퍼 + 우솝 보잉열도 + 카포네 갱 벳지 + 해적선 방주맥심조합 (maxim)
레드포스호 샹크스 + 버기 마기탄 + 해적선 레드포스호조합 (redforce)

초월 유닛

이름 재료 타이핑 (영문)
루피 루피 JET개틀링 + 루피 나이트메어 + 와이퍼 + 초월 쿠마 해적왕이될사나이 (luffy tr)
조로 조로 + 검호 미호크 + 바르토로메오 + 페로나 + 초월 쿠마 최강의검사 (zoro tr)
나미 나미 + 금사자 시키 + 나미 크리마텍트 + 아론 + 초월 쿠마 날씨는맑음 (nami tr)
우솝 카르가라 + 슈가 + 벤베크만 + 페로나 + 초월 쿠마 바다의전사 (usopp tr)
상디 상디 + 크래커 + 나미 크리마텍트 + 초월 쿠마 지옥에서돌아온자 (sanji tr)
쵸파 울티 + 히루루크 + 쵸파 + 초월 쿠마 순록의사! (chopper tr)
로빈 바르토로메오 + 로브 루치 + 크로커다일 + 로빈 + 초월 쿠마 악마의자식 (robin tr)
프랑키 쿠마 폭군 + 베이비5 + 카쿠 + 파이러츠도킹5 + 초월 쿠마 무적의아이언파이러츠 (franky tr)
브룩 라분 + 류마 + 페로나 네거티브프린세스 + 모리아 + 초월 쿠마 소울킹 (brook tr)
샹크스 샹크스 + 검호 미호크 + 캡틴 키드 + 루피 기어세컨드 + 초월 쿠마 정상해전종결자 (shanks tr)
검은수염 검은수염 + 흰수염 + 바질 호킨스 + 초월 쿠마 치밀하고비열한어둠 (tichi tr)
시라호시 징베 + 반더데켄 + 비비 + 초월 쿠마 해왕류를다스리는인어공주 (shirahoshi tr)
아오키지 제파 + 아오키지 + 로빈 오하라 + 타시기 + 초월 쿠마 전해군대장푸른꿩 (aokiji tr)
아카이누 센고쿠 + 아카이누 + 에이스 2번대대장 + 루치 + 초월 쿠마 해군원수붉은개 (akainu tr)
키자루 후지토라 + 레일리 + 키자루 + 아론 + 바질 호킨스 + 초월 쿠마 해군대장노란원숭이 (kizaru tr)
로우 로우 + 시저 + 베포 * 2 + 초월 쿠마 백개의심장을바친자 (law tr)
도플라밍고 도플라밍고 + 베이비5 + 베르고 + 슈가 + 저격왕 + 초월 쿠마 어둠의조커 (joker tr)
사보 마르코 환수종불사조 + 사보 + 바제스 + 에이스 + 해적선 + 초월 쿠마 혁명군참모총장 (sabo tr)
후지토라 후지토라 + 킨에몬 + 모몬가 + 타시기 + 초월 쿠마 분노하는맹호 (fujitora tr)
타시기 코비 + 시저 + 스모커 + 타시기 + 초월 쿠마 해군의홍일점 (tashigi tr)
루치 로브 루치 + 사보 + 로빈 오하라 + 초월 쿠마 검은정의를좇는하얀새 (lucci tr)
바질 호킨스 조로 + 코비 + 키자루 + 스트로맨 + 초월 쿠마 운명의마술사 (hawkins tr)
징베 징베 + 라분 + 에이스 2번대대장 + 초월 쿠마 바다의협객 (jinbe tr)
스네이크맨 루피 나이트메어 + 레드포스호 + 사보 + 에이스 + 초월 쿠마 다섯번째황제 (snakeman tr)
키드 네코마무시 + 킬러 + 로우 + 루피 기어세컨드 + 키드 초신성 + 초월 쿠마 신세대의이단아 (kid tr)
야마토 울티 + 토키 + 핸콕 + 나미 크리마텍트 + 에이스 2번대대장 + 초월 쿠마 오니히메 (yamato tr)
료쿠규 료쿠규 + 후지토라 + 루피 기어세컨드 + 초월 쿠마 신해군대장초록소 (ryokugyu tr)

버기 나미 + 이완코브 + 샹크스 + 마젤란 + 버기 마기탄 + 봉쿠레 + 압살롬 + 클리어 횟수 10회 + 목재 5개 천냥광대 (buggy et)

우타 샹크스 + 모리아 + 봉쿠레 + 레일리 + 브룩 음악가 + 루피 + 우타의 헤드셋 + 클리어 횟수 30회 + 확장팩 + 목재 5개 세계의가희 (u-ta et)
