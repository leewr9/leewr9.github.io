조합법

스페셜 유닛

이름 재료 타이핑 (영문)
카부토 앙코 + 목재 8 -카부토 구매 (-kabuto buy)
토비 나가토 + 목재 8 -토비 구매 (-tobi buy)

히든 유닛

이름 재료 타이핑 (영문)
가마키치 가마분타 + 가마히로 가마키치 조합 (gamakichi)
나루토 6꼬리 나루토 풍둔 + 나가토 6꼬리 조합 (sixtails)
마키 가아라 + 카카시 마키 조합 (maki)
센쥬 하시라마 야마토 + 이타치 하시라마 조합 (hashirama)
센쥬 토비라마 나루토 풍둔 + 앙코 토비라마 조합 (tobirama)
테마리 풍둔술사 텐텐 + 칸쿠로 + 칸쿠로 테마리 조합 (temari)
쿠로츠치 사이 + 데이다라 쿠로츠치 조합 (kurotsuchi)
아카츠치 텐텐 + 흰 제츠 아카츠치 조합 (akatsuchi)
키츠치 이노 + 오모이 키츠치 조합 (kitsuchi)
토루네 아부라메 시노 + 야마나카 후 토루네 조합 (torune)
오로치마루 백사 오로치마루 + 쥬고 주인화 오로치마루 조합 (orochimaru)
도롱뇽의 한조 앙코 + 지라이야 + 츠나데 한조 조합 (hanzo)
카부토 세포 이식 앙코 + 쿠레나이 스파이 조합 (spy)
시무라 단조 카카시 + 야마나카 후 단조 조합 (danzo)
쥬고 완전주인화 쥬고 주인화 + 쥬고 주인화 쥬고 조합 (jugo)
히나타 히나타 + 아부라메 시노 + 카부토 히나타 조합 (hinata)
사소리 완전변형 사소리 아카츠키 + 칸쿠로 사소리 조합 (sasori)
지라이야 전설의 3닌자 가마분타 + 지라이야 지라이야 조합 (jiraiya)
시이 킬러비 + 오모이 시이 조합 (shii)
키바 쌍두랑 쿠레나이 + 휴우가 네지 + 키바 키바 조합 (kiba)
사스케 기린 사스케 아카츠키 + 이타치 아카츠키 기린 조합 (kirin)
아오 키사메 + 쵸쥬로 아오 조합 (ao)
안개마을 닌자도 7인방 자부자 + 록리 연화 + 쵸쥬로 닌자도 조합 (ninjado)

리미트 유닛

이름 재료 타이핑 (영문)
오오츠츠키 인드라 마다라 + 사스케 + 토비 + 클리어 점수 500점 육도선인의 눈을 물려받은 자 (otsutsuki indra)
오오츠츠키 아수라 나루토 선인모드 + 센쥬 하시라마 + 클리어 점수 500점 육도선인의 육체를 물려받은 자 (otsutsuki asura)

영멸 유닛

이름 재료 타이핑 (영문)
위장 스사노오(나루토) 나루토 미수화 + 400 골드 + 인피니티 위습 2개 + 우즈마키 나루토 - 7대 호카게 + 우치하 사스케 - 방랑 닌자 두 사람의 유대감 (majestic susanoo naruto)
위장 스사노오(사스케) 사스케 3단계 스사노오 + 400 골드 + 인피니티 위습 2개 + 우즈마키 나루토 - 7대 호카게 + 우치하 사스케 - 방랑 닌자 또 한명의 호카게 (majestic susanoo sasuke)
오오츠츠키 모모시키 오오츠츠키 모모시키 + 차크라 열매 단 + 오오츠츠키 킨시키 + 차크라 컨트롤 - 음 신목의 부활을 꾀하는 자 (otsutsuki momoshiki)
오오츠츠키 카구야 10미 + 검은 제츠 * 2 + 미수 차크라 나의 의지는 카구야다 (otsutsuki kaguya)
