저는 업무 자동화와 데이터 처리 효율화에 깊은 관심을 가지고, Python과 Shell 기반의 운영 스크립트부터 Airflow를 활용한 안정적이고 확장 가능한 데이터 파이프라인 구축까지 다양한 경험을 쌓아왔습니다. 또한, AWS와 GCP 클라우드 환경에서 인프라를 운영하며 Jenkins와 GitHub Actions를 이용한 CI/CD 자동화를 통해 서비스의 안정성과 배포 효율성 향상에 기여해왔습니다. 웹 서비스 개발 분야에서는 Django와 Flask를 활용한 REST API 설계 및 구축에 풍부한 경험을 갖추고 있으며, LangChain 기반 AI 응용 서비스와 PySide를 활용한 내부 GUI 툴 개발까지 폭넓은 영역을 다루고 있습니다. PostgreSQL, MongoDB 데이터 모델링과 웹 크롤링 및 API 연동을 통한 데이터 수집 자동화도 자연스럽게 수행해왔습니다. 저는 단순히 코드를 작성하는 개발자를 넘어, 전체 시스템의 흐름을 이해하며 운영 효율성과 실제 사용성을 고려한 개선에 주력하는 개발자입니다. 협업과 원활한 커뮤니케이션을 중요하게 생각하며, 새로운 기술과 도구를 빠르게 습득하고 적용하는 데 자신이 있습니다. 변화하는 환경에 유연하게 대응하며 주도적으로 개선을 이끌 수 있는 조직에서, 저의 경험과 열정을 바탕으로 함께 성장해 나가고 싶습니다. 감사합니다.
